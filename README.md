# Hrishikesh Thorat Portfolio

A personal portfolio built with Next.js, React, Tailwind CSS, Framer Motion,
and static export support for simple hosting.

## Highlights

- Responsive one-page portfolio with hero, projects, about, skills, contact,
  and footer sections
- Project cards with screenshots, stack tags, and GitHub links
- Resume download from `public/Resume.pdf`
- Contact form that opens a prefilled email draft
- Dark/light theme toggle with saved preference

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS 4
- Framer Motion
- React Icons
- tsParticles

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Build

```bash
npm run build
```

The project is configured with `output: "export"` in `next.config.mjs`.
